# Python - Object-relational mapping

This project explores connecting to MySQL databases from Python using:
1. MySQLdb - raw SQL queries
2. SQLAlchemy - Object Relational Mapping (ORM)

## Tasks
- Task 0: Get all states from database using MySQLdb

## Requirements
- Python 3.8.5
- MySQLdb 2.0.x
- SQLAlchemy 1.4.x
- MySQL 8.0

## Usage
```
./0-select_states.py root root hbtn_0e_0_usa
```
